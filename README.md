## Installation instructions

1. Have the Gatsby CLI (gatsby-cli) installed globally by running: `npm install -g gatsby-cli`
1. `npm install`
1. `npm run develop`
1. Open http://localhost:8000/

### Suggestions

1. Use `gatsby clean` if you add new folders
